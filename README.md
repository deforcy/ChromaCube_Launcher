# ChromaCube Launcher

The official ChromaCube launcher. It is a simple app that connects you to the
ChromaCube Minecraft servers with a single click.
</p>

## Getting started

1. Open the launcher.
2. If you are asked for an access code, paste the personal code you were given.
3. Click **Connect to network** and wait until it says **Connected**.
4. Open Minecraft, go to **Multiplayer**, and add the server using the address
   shown in the launcher.
5. Join and play.

That's it. When you are done you can close the launcher, and it keeps running
quietly in the system tray so you can reconnect any time.

## Good to know

- The launcher shows your live ping to each server.
- It keeps itself up to date automatically.
- Closing the window minimizes it to the tray. To quit fully, right-click the
  tray icon and choose Close.

## Need help?

If something is not working, reach out to the server owner.
